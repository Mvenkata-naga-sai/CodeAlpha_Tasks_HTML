# 📌 Project Management Tool
---
A simple Trello/Asana-like project management application where users can create projects, manage tasks, and collaborate efficiently.
--
### 🚀 Features: 
---
-> User Authentication (Register/Login)
-> Create and Manage Projects
-> Add and Assign Tasks
-> Comment on Tasks
-> Team Collaboration Interface
--

### ⚙️ Backend Setup (Node.js):
---
* Step 1: Navigate to backend folder
cd backend
* Step 2: Install dependencies
npm install
* Step 3: Start the server
node server.js
--
### 🌐 Frontend Setup:
---
Step 1: Open frontend folder

Go to:

frontend/index.html
Step 2: Run the application
Double-click index.html
OR
Use Live Server in VS Code
--
### 🔗 API Connection
---
Frontend connects to backend at:

http://localhost:5000
--
### ⚠️ Make sure backend is running before using frontend.
---
### 🧪 How to Use:
-> Register a new user
-> Login to your account
-> Create a project
-> Add tasks to the project
-> Assign tasks
-> Add comments for communication
--

### ✅ Conclusion
---
This Project Management Tool demonstrates the fundamental features of modern collaboration platforms like Trello and Asana. It enables users to organize projects, manage tasks, and communicate effectively within a team.

The project follows a simple full-stack architecture, making it easy to understand and extend. Although it currently uses basic storage and limited features, it provides a strong foundation for building more advanced and scalable applications.

Overall, this project helps in gaining practical knowledge of full-stack development, including frontend-backend integration, user authentication, and task management workflows.
